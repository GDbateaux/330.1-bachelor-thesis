# Examples

Compile and run any example with:

```bash
swift run Carl examples/game-of-life.carl -o game-of-life
./game-of-life
```

| File | Description |
|------|-------------|
| `excitable-medium-hex.carl` | Excitable medium on hexagonal grid |
| `excitable-medium.carl` | Excitable medium |
| `forest-fire-3d.carl` | Forest fire in 3D |
| `forest-fire.carl` | Forest fire simulation |
| `game-of-life.carl` | Conway's Game of Life |
| `wireworld.carl` | Wireworld simulation |
